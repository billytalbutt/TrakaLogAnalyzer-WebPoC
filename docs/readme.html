What Is This?

The Traka Log Analyzer is a desktop application purpose-built for Traka support engineers. It reads, parses, and analyses log files from the Traka product suite ‚Äî Business Engine, Comms Engine, Integration Engine, and all integration plugins ‚Äî then surfaces issues, suggests fixes, and lets you compare files side by side.

It runs locally on your machine. No data is sent anywhere. No internet connection is required after installation.

Installation & Running

Option A: From Pre-Built Installer

If you have the .exe installer (built with Electron Builder), simply run it. The application installs to your user profile and creates a Start Menu shortcut.

Option B: From Source (Development)

  1. Install Node.js (v18 or later)
  2. Open a terminal in the project directory
  3. Run npm install to install dependencies
  4. Run npm start to launch the application

Option C: Portable (No Install)

Copy the deployment folder to any Windows machine with the Electron runtime available and run npm start, or use the pre-packaged portable build if available.

Auto-Discovery Paths

When using Scan Traka Directories, the following locations are searched:

C:\Program Files\Traka\Traka Business Engine Service\Support\Logs
C:\Program Files\Traka\Traka Comms Engine Service\Support\Logs
C:\inetpub\wwwroot\TrakaWeb\App_Data\Support\Logs
C:\ProgramData\Traka\Logs

Additional custom directories can be configured in Settings.
