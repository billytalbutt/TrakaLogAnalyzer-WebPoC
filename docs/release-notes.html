Release Notes

Traka Log Analyzer

A history of every release, what changed, what was fixed, and what's new.

Version 3.0.0 — Desktop Edition

February 2026

NEW Electron Desktop Application

  • Full desktop application built on Electron — runs natively on Windows
  • Automatic Log Discovery — scans standard Traka installation directories and loads the newest logs from each engine
  • Custom Directory Scanning — pick any folder and recursively find all log, text, and config files
  • Native File Watching — Live Tail powered by Chokidar for real-time file monitoring with byte-level precision
  • File rotation and truncation detection for Live Tail

NEW Compare Page Enhancements

  • Time Sync — click any timestamped line and all panels jump to that moment. Adjustable time window (1–60 seconds)
  • Minimize Panels — collapse individual panels to a floating dock; remaining panels fill the space
  • Search Filter Mode — toggle between highlighting matches and filtering to show only matching lines
  • Highlight Differences info dialog — explains the feature's intended use before running
  • Improved search highlight visibility with inline text marking
  • Search clears properly when input is emptied
  • Loading animations for Time Sync enable/disable

NEW Highlight Rules

  • BareTail-style custom colour rules for keywords and phrases
  • Choose text colour and background colour with live preview
  • Apply rules per file or across all loaded files
  • Accessible from both Log Viewer and Compare page toolbars
  • Rules persist in browser storage between sessions

NEW FAQs / Knowledge Base

  • Searchable knowledge base of 50+ known Traka issues and resolutions
  • Category filtering (Configuration, Security, Database, Communication, etc.)
  • "Detected in Logs" badges when issues match your loaded files
  • Step-by-step resolution instructions with prerequisites

NEW Collapsible Sidebar

  • Click to collapse the sidebar to icons only — content area scales to fill the space
  • Tooltip labels on hover when collapsed
  • State persists between sessions

NEW User Guide

  • Comprehensive HTML user guide covering every feature
  • Branded design matching the application's visual identity
  • Print-friendly for PDF export

IMPROVED UX Polish

  • Loading animations for fullscreen toggle, Time Sync, and Highlight Rules modal
  • Time Sync panel collapse now reclaims space for log panels (smooth height transition)
  • Centred Auto-Discovery section on Home page
  • Stitch Files button uses neutral grey to reduce visual noise

FIX Bug Fixes

  • Log Viewer now displays the first file correctly on initial load (was showing empty viewer)
  • Highlight Rules modal now opens reliably from the Compare page
  • Compare search highlights clear when the search field is emptied
  • Minimised panels dock no longer clipped by overflow on compare container

Version 2.1 — Time Sync & Stitching

January 2026

NEW Features

  • Time Sync — initial implementation for the Compare page
  • File Stitching — merge multiple logs into a single chronological timeline
  • Date Sorting — sort log lines by timestamp (original order, oldest first, newest first)
  • Go to Line — jump to any line number in the viewer

IMPROVED Improvements

  • Compare page now supports drag-and-drop file loading
  • Fullscreen mode for both Viewer and Compare pages
  • Performance optimisations for large files (virtualised rendering)

Version 2.0 — Compare & Analytics

January 2026

NEW Features

  • Compare Page — side-by-side comparison of up to 6 files
  • Analytics Dashboard — charts for issue distribution, trends, and recommendations
  • Sync Scroll — lock all compare panels to scroll together
  • Highlight Differences — mark unique and common lines across files
  • PDF Report Export — generate shareable issue reports

Version 1.0 — Initial Release

January 2026

NEW Features

  • Single-file log viewer with colour-coded log levels
  • Search with regex support and match navigation
  • Log level and engine type filtering
  • Automatic issue detection with severity ratings
  • Issues page with category filters and detail modal
  • Drag-and-drop and file picker for loading files
  • Dark theme matching the TrakaWEB Tools Suite style
