Permitted Use

This software is provided by Traka for use by authorised support engineers, integrators, and customers. You may install it on any machine where Traka products are deployed or being supported.

What This Application Does

The Traka Log Analyzer reads log files from Traka Business Engine, Comms Engine, Integration Engine, and integration plugins. It provides real-time log monitoring, side-by-side file comparison with time synchronisation, automatic detection of 50+ known issue patterns, and a searchable knowledge base of solutions.

Data Privacy

This application runs entirely on your local machine. No log data is transmitted over any network. No telemetry or analytics are collected.

System Requirements

Windows 10 (64-bit) or later. 4 GB RAM minimum (8 GB recommended). 250 MB disk space.
